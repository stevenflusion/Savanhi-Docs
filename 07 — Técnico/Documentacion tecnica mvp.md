# # Introduccion

En la presente página se describirá el contenido de la documentación técnica en base al estándar de desarrollo de software C4, con el objetivo de describir cada parte técnica con el fin de que cada integrante o a futuro cualquier person adentro del proyecto pueda integrarse de forma rápida.

El objetivo de esto es tener claro el alcance de este MVP, para no ir con ideas descabelladas y aleatorias sino tener un único objetivo que alcanzar.
